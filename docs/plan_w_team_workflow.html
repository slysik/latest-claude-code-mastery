ü§î Can I use /plan_w_team for ANY task?

  • Yes! It's a general-purpose planning workflow for any engineering task
  • Best for complex tasks that benefit from parallel work streams
  • Great for multi-file changes across your codebase
  • Perfect when you want validation after implementation
  • Works for: feature development, refactoring, migrations, documentation, testing
  • Overkill for: single-file fixes, quick questions, simple changes
